# 🎓 College Event Registration System

## 📌 Overview
This is a full-stack web application where students can register for college events like Tech Fest, Cultural Night, and Workshops.

The project includes a modern frontend UI and a backend server to store registration data.

---

## 🚀 Features
- Professional modern UI (Neon + Glass design)
- Event registration form
- Data stored in database
- Success message after submission
- Fully responsive design
- Public frontend deployment

---

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Node.js
- Express.js
- SQLite Database
- GitHub Pages (Frontend Hosting)
